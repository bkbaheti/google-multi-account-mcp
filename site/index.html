MCP Google Multi-Account

A local MCP (Model Context Protocol) server that gives AI assistants like Claude secure access to your Google accounts. Connect multiple Google accounts and work with Gmail, Google Drive, and Google Calendar — all from a single server running on your machine.

Quick Start

Claude Code CLI

Add the server in one command:

claude mcp add google-multi-account -- npx -y @anthropic/mcp-google

Claude Desktop

Add the following to your Claude Desktop configuration file (claude_desktop_config.json):

{
  "mcpServers": {
    "google": {
      "command": "npx",
      "args": ["-y", "@anthropic/mcp-google"]
    }
  }
}

On first use, the server will prompt you to provide your own Google OAuth client credentials and guide you through connecting your Google account.

Features

  • Multi-account support — Connect and switch between multiple Google accounts seamlessly
  • Gmail — Search, read, draft, send, reply, manage labels, archive, and trash messages
  • Google Drive — Search, list, read, upload, copy, move, rename, share, and manage files
  • Google Calendar — List, search, create, update, delete events, check availability, and RSVP
  • Zero-config — Install and run with a single npx command, no build step required
  • Local tokens — OAuth tokens are stored only on your machine at ~/.config/mcp-google/
  • Safety gates — Destructive actions like sending email or sharing files require explicit confirmation
  • Tiered scopes — Request only the Google API permissions you need
